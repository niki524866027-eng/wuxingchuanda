# 天机 · Cyber Wuxing 女性限定版

一个基于五行理论的赛博风格穿衣指南网站，专为女性用户设计。

## 功能特点

- 🎨 **赛博美学设计** - 霓虹风格界面，玻璃态卡片效果
- 🌿 **五行穿衣指南** - 根据日期计算五行属性，推荐最佳穿搭颜色
- 📅 **日期选择** - 支持查看今日、明日、后日的穿衣建议
- 🎯 **宜忌事项** - 显示当日宜办和忌办事项
- 🎵 **背景音乐** - 沉浸式体验
- 📱 **响应式设计** - 完美适配移动端和桌面端

## 技术栈

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript (原生)
- GitHub Pages 部署

## 使用方法

1. 选择日期或使用快捷按钮（今日/明日/后日）
2. 查看推荐的五行颜色搭配
3. 参考幸运配饰建议
4. 查看当日宜忌事项

- **木** (Wood) - 绿色系，生机感，温柔系
- **火** (Fire) - 红色/粉色系，热烈感，元气系
- **土** (Earth) - 黄色/咖色系，稳重感，优雅系
- **金** (Metal) - 白色/金银色系，冷冽感，御姐系
- **水** (Water) - 黑色/蓝色系，静谧感，清冷系
